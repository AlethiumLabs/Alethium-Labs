# Alethium Labs

Official landing site for Alethium Labs, an AI R&D and systems design lab.

## Structure

- `index.html`: Main entry point (Split screen layout).
- `assets/css/main.css`: Custom styles and glassmorphism utilities.
- `assets/js/main.js`: GSAP animations and interactions.
- `seo/llms.md`: Profile for AI agents and LLMs.

## Stack

- HTML5
- Tailwind CSS (via CDN)
- GSAP (via CDN)
- Vanilla JS
