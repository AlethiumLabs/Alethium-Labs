Product Overview

Alethium Vault is a professional-grade password manager built for people who do not trust cloud-first security models. Your credentials are stored locally, encrypted using AES-256-GCM, and cryptographically bound to your device’s physical hardware using TPM-backed key protection.

Even if your vault file is copied or stolen, it cannot be decrypted on another machine. Alethium Vault includes Foundry, a high-entropy password generator, and a built-in offline Knowledge Base for storing security protocols and operational notes.

Free to try during early access, and designed for complete data sovereignty.

Security Architecture

Security is not a feature. It is the foundation.

Alethium Vault uses Scrypt for key derivation with high-cost parameters and AES-256-GCM for authenticated encryption. To go further, the vault’s master secret is wrapped using TPM-backed RSA-OAEP-SHA256, binding the vault cryptographically to the physical device it was created on.

The cryptographic design prioritizes transparency, mathematical soundness, and long-term compatibility.

This Means

  • Stolen vault files are useless off-device
  • No hidden backdoors or proprietary crypto
  • Fully offline, verifiable security primitives

Foundry Password Generator

Foundry is Alethium Vault’s built-in high-entropy password generator.
